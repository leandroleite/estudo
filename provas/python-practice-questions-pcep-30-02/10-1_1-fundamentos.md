# QUESTAO NUMERO 1

Qual é a função principal de um interpretador em Python?

- (A) Compilar todo o código antes da execução
- (B) Executar código linha por linha

Um interpretador em Python executa código linha por linha, o que permite execução imediata e depuração mais fácil, mas pode ser mais lento para programas maiores em comparação com linguagens compiladas.
