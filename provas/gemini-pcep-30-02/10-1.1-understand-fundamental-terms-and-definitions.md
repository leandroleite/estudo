# QUESTAO NUMERO 1

## Synopsis: Classificação da Língua

Qual das seguintes opções melhor descreve o Python?

- (A) Uma linguagem de programação de baixo nível
- (B) Uma linguagem de programação compilada
- (C) Uma linguagem de programação interpretada de alto nível

## Answer: C

## Explicação

# QUESTAO NUMERO 2

## Synopsis: Tipos de Erro

Um programador escreve um script, mas durante a execução, o programa para porque tenta dividir um número por zero. Este é um exemplo de violação de:

- (A) Lexis
- (B) Sintaxe
- (C) Semântica
- (D) Compilação

## Answer: C

## Explicação

Dividir por zero é um **erro semântico — a "gramática" (sintaxe) é válida, mas o "significado" ou lógica é inválido.**

# QUESTAO NUMERO 3

## Synopsis: O Papel do Intérprete

Qual é a função principal do interpretador de Python?

- (A) Converter todo o código-fonte em um arquivo .exe antes de executar.
- (B) Ler e executar o código-fonte linha por linha.
- (C) Para verificar o código apenas para erros de ortografia.

## Answer: B

## Explicação

Diferente de um compilador, o interpretador processa o código sequencialmente.

# QUESTAO NUMERO 4

## Synopsis: O Processo de Compilação

No contexto do Python, qual é o nome específico do código intermediário gerado pelo interpretador antes de ser executado pela Máquina Virtual?

- (A) Código de máquina
- (B) Bytecode
- (C) String binária
- (D) Fragmento de Origem

## Answer: B

## Explicação

Python compila o código-fonte em **bytecode** (arquivos .pyc) que a Máquina Virtual Python (PVM) então executa.

# QUESTAO NUMERO 6

## Synopsis: Linguagens de Programação Natural vs. Programming

Qual das seguintes características é que **as linguagens de programação** compartilham com **as línguas naturais**?

- (A) Ambos não possuem um conjunto formal de regras semânticas.
- (B) Ambos usam um alfabeto definido e um conjunto de símbolos (Lexis).
- (C) Ambos evoluíram naturalmente, sem intervenção humana.
- (D) Ambos permitem total ambiguidade nas instruções.

## Answer: B

## Explicação

Ambos os tipos de língua dependem do **Lexis** (um dicionário de símbolos/palavras). No entanto, as linguagens de programação devem ser inequívocas, ao contrário das linguagens naturais.

# QUESTAO NUMERO 7

- (B) Esquecer um parêntese de encerramento em uma função print().
- (C) Usando um nome de variável que ainda não foi definido.
